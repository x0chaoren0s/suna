# pdf转md工具
## 1. Marker ⭐最强（针对论文）
```bash
# 开源工具，专门优化学术论文转换
pip install marker-pdf
marker-pdf input.pdf -o output.md
```
-✅ 保留公式（转为LaTeX）
-✅ 保留表格结构
-✅ 图片转为占位符（可手动添加描述）

## 2. Mathpix Snip ⭐付费但精准
- 对数学公式识别极其准确
- 截图即可转换为LaTeX或Markdown
- 适合手动处理重要论文

## 3. Pandoc
```bash
pandoc paper.pdf -o paper.md
```
- 免费开源
- 通用格式转换
- 对复杂PDF效果一般

## 4. ChatGPT/Claude + PDF 🤖智能方案
```bash
上传PDF到ChatGPT，提示词：
"请将这篇论文转换为Markdown格式，保留所有公式（使用LaTeX），
并为所有图表添加详细的文字描述。"
```
- 准确度最高（AI理解内容）
- 会自动补充图表描述
- 但需要手动操作

## 💡 最佳实践建议
我的推荐策略（平衡质量和效率）：
1. 重要论文（核心参考文献）
转成Markdown ✅
手动补充关键图表描述 ✅
质量最高
2. 一般参考文献
直接上传PDF ⚠️
接受文本提取的不完美
节省时间
3. 数学/公式密集论文
必须转Markdown ❗
使用Marker或Mathpix
否则公式全部乱码
4. 扫描版PDF
先OCR（使用Adobe Acrobat或ABBYY FineReader）
再转Markdown
否则完全无法提取
